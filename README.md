# דאשבורד אינטרקטיבי — פרחי איריס ורקפת

אפליקציית ווב אינטראקטיבית לחקירת מאפייני פרחים, בנויה עם **Dash** ו-**Plotly**.

## תיאור

הפרויקט מציג גרף פיזור (scatter plot) דינמי שמאפשר לבחור תכונות לציר X ולציר Y מתוך מערך הנתונים. הנתונים כוללים את מערך האיריס הקלאסי (Iris) יחד עם נתוני דוגמה לפרח רקפת (cyclamen) — סוג רביעי שנוסף ידנית. הצבעים בגרף מייצגים את סוגי הפרח השונים.

## תכונות

- בחירת תכונה לציר X מתוך רשימה נפתחת
- בחירת תכונה לציר Y מתוך רשימה נפתחת
- עדכון מיידי של הגרף בעת שינוי הבחירה
- צביעה לפי סוג פרח (`species`)
- נתוני דוגמה לפרח רקפת (10 רשומות עם מאפיינים דומים לאיריס)
- איחוד מערכי נתונים — איריס ורקפת — באמצעות `pd.concat`
- ממשק בעברית

## דרישות מערכת

- Python 3.8 ומעלה
- החבילות: `dash`, `plotly`, `pandas`

## איך זה עובד

1. נתוני האיריס נטענים מ-`plotly.express.data.iris()`.
2. נוצר מילון `cyclamen_data` עם 10 רשומות לפרח רקפת (אורך/רוחב עלה וכותרת, ועמודת `species`).
3. הנתונים מאוחדים ל-DataFrame אחד: `pd.concat([px.data.iris(), cyclamen_df], ignore_index=True)`.
4. שני רכיבי `dcc.Dropdown` מאפשרים לבחור עמודות מספריות לצירים.
5. callback של Dash מעדכן את `dcc.Graph` בכל שינוי בבחירה.
6. `plotly.express.scatter` יוצר את הגרף עם צביעה לפי `species`.
